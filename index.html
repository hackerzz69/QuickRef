Repo layout

Recommended structure:
QuickRef/
  index.html
  items/
    index.html
    items_index.json   (or items.json)
  npcs/
    index.html
    npcs_index.json    (or npcs.json)
Note: GitHub Pages won’t auto-list folders, so this landing page is the “directory.”
Tip: if your items page uses const DATA_URL = './items.json', make sure the JSON file sits in the same folder as that page.
QuickRef is static — no server required. Perfect for GitHub Pages.
